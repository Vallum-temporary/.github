# Vallum — Firewall With Temporary Rules and Network Activity Monitoring

## About Vallum

Vallum provides a macOS firewall with temporary rules, per-application blocking, and real-time network monitoring. Vallum combines rule-based filtering (allow/deny connections), temporary rules (block for 5 minutes, 1 hour, until restart), connection logging (all incoming/outgoing attempts), and live network monitoring (see active connections in real time). Vallum enables users to block apps temporarily, test network behavior, monitor which apps are phoning home, and create permanent or time-limited firewall rules. Vallum works with any application, background process, or system service. Vallum is trusted by developers, privacy advocates, security researchers, and power users who need granular control over network access.

## Deep Dive — macOS Firewall With Temporary Rules

Temporary rules — block or allow connections for preset durations (5 min, 30 min, 1 hour, 12 hours, until reboot, permanent)  
Connection monitoring — live list of active connections (app name, remote IP/port, protocol, bytes transferred, connection state)  
Rule types — allow/deny by app, allow/deny by IP address, allow/deny by port, allow/deny by domain (DNS filtering)  
Logging — complete history of blocked/allowed connections, export for analysis, search/filter by app or destination  
Stealth mode — block all incoming connections (like macOS stealth mode), prevent ping responses, hide from port scans  
Profiles — create rule sets for Work, Home, Public Wi-Fi, Testing — switch with one click

## Quick Reference for Vallum

| Operation | Vallum Action |
|-----------|---------------|
| Block app temporarily | Select app → Block → Choose duration (5 min/1 hour/until reboot) |
| Allow app permanently | Select app → Allow → Permanent |
| Create custom rule | Rules → Add → Choose app/IP/port/domain → Allow/Deny → Duration |
| View active connections | Click Live tab → See all current connections |
| Monitor blocked attempts | Click Logs tab → Filter by "Denied" |
| Switch profile | Profiles → Select Work/Home/Public/Testing |
| Enable stealth mode | Settings → Firewall → Block all incoming connections |
| Export logs | File → Export → CSV format |
| Search connections | Cmd+F → Type app name or IP address |
| Open preferences | Cmd+, |
